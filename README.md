# Online_Coup

5 personagens padrões:
  Banqueiro;
  Ladrão;
  Médico;
  Assassino;
  Diplomata;
Gerar duas cartas aleatórias para cada jogador no início da partida;
Exibir uma lista com as ações que o jogador pode fazer no turno dele;
Quando escolhe uma ação, aparece uma tela resumindo a ação:
Pra quem fez a ação aparece só o que ele escolheu;
Pra todos os outros aparece a ação e um botão “Questionar”
OBS. para ações mais especificas como assassinato precisa planejar melhor como vai funcionar, pois há a contra-ação do médico por exemplo
